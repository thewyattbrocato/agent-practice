# Project Agent Instructions

## Project overview

This is a beginner practice repository for learning agentic engineering workflows.

## Agent behavior

- Explain concepts clearly.
- Do not make large changes without a plan.
- Do not commit unless explicitly asked.
- Do not push unless explicitly asked.
- Prefer small, safe, educational changes.
- Prefer concise, token-efficient outputs.

## Tool preferences

- Use ripgrep for search.
- Use git diff for reviewing changes.
- Use gh or gh-axi for GitHub operations when helpful.

## Verification

For now, verify changes by reading the files and checking `git diff`.
